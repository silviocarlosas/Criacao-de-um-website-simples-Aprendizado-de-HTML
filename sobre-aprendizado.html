Como esta página foi feita

Esta página foi criada à partir de vídeoaulas postadas no Youtube, usando a linguagem de marcação HTML, apenas para aprendizado. Inicialmente foram ensinados o conceito de HTML e depois o significado e uso das principais TAGS. Na página de "Contato" ainda será necessário assistir mais aulas. Na sequência do HTML, aprenderei CSS e Javascript para o back-end. Lembrando que ainda falta mencionar o uso de todas as imagens deste site e seus respectivos usos de imagem.
